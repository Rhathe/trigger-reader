thehackerati-reader
===================



Third-Party Software
--------------------

* trigger.io
* backbone.js
* zepto.js
* underscore.js

Get the Code
------------

    % git clone

Build It
--------

    % forge build

Running
-------

To launch the app in the iOS Simulator (requires iOS Simulator):

    $ forge run ios

To launch the app in the default browser (requires Node.js):

    $ forge run web
